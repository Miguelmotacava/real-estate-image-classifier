# Invitar profesores al proyecto W&B

El acceso al workspace se gestiona desde la web de Weights & Biases (la API
pública no expone una operación de invitación). Pasos a seguir:

1. Entrar en <https://wandb.ai/jumipe_meflipapesos/real-estate-classifier>.
2. Ir al menú **Settings → Members** (icono de engranaje arriba a la derecha,
   o ruta `/settings/team` si es entity de tipo team).
3. Pulsar **Invite a new member** y añadir los siguientes correos como
   *Viewer* (acceso de solo lectura suficiente para revisar la práctica):
   - `agascon@comillas.edu`
   - `rkramer@comillas.edu`
4. Confirmar el envío. Las personas invitadas recibirán un correo con el enlace
   directo al workspace.

> Nota: con un proyecto **público** no es estrictamente necesaria la invitación
> para visualizar runs y reports, pero se incluye para garantizar el acceso a
> todas las pestañas (Sweeps, Workspace personal, Artifacts).
